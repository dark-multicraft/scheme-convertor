# WorldEdit Converter

> Fork of [multi-nono/3D-Model-To-world.edit-command-minetest-W.E-](https://github.com/multi-nono/3D-Model-To-world.edit-command-minetest-W.E-)

**What can it do? üßê**  
Converts various file types into WorldEdit command format.  
Can be executed on any server with mesecons and WorldEdit installed!  

**Supported Files**  
> Minecraft Schematic Files (`.schem`, `.nbt`)  
> Minetest WorldEdit Files (`.we`)  
> 3D Model Files (`.obj`, `.stl`)  

**Usage üìñ**  
1. Prepare your schematic.  
2. Open the [Converter Site](https://dark-multicraft.github.io/WorldEdit-Converter/).  
3. Copy the commands generated by the converter site.  
4. Paste the copied commands into a Command Block in your world.  
5. Select 1 block using `//1` and `//2` at the location where you want to place it.  
6. Execute the Command Block.  

**Warning ‚ö†Ô∏è**  
- Depending on the size of the object, it may be quite heavy/laggy, so please paste at your own risk.  
- Selecting a wide area with `//1` and `//2` will cause the schematic to distort.  
- Once placed, it cannot be removed with `//undo`, so please test in a safe area first.  
- When using distributed assets, please strictly follow the original terms of use.
